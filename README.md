Sistema de Ventas e Inventario - BUFFET TIEN LAN ZU SHI

Descripción

Sistema web desarrollado para automatizar la gestión de ventas, inventario y clientes del Buffet Vegetariano Tien Lan Zu Shi, mejorando el control de la información y optimizando los procesos del negocio.

Características

- Gestión de clientes.
- Administración de platillos.
- Control de inventario.
- Registro de ventas.
- Dashboard con indicadores.
- API REST y base de datos SQL Server.

Tecnologías Utilizadas

Frontend

- HTML
- CSS
- JavaScript

Backend

- C#
- ASP.NET Core Web API

Base de Datos

- SQL Server

Herramientas

- Visual Studio
- Visual Code
- GitHub
- Power BI

Arquitectura

El sistema está basado en una arquitectura de tres capas:

1. Presentación (Frontend)
2. Lógica de Negocio (API REST)
3. Datos (SQL Server)

Beneficios

- Automatización de procesos.
- Mejor control de inventario.
- Reducción de errores.
- Centralización de la información.
- Apoyo a la toma de decisiones.
